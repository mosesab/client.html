Project Documentation: Fortal Browser

Introduction

Imagine having a secure and controlled workspace right in your browser. Fortal Browser is designed to do exactly that. It allows you to create virtual, isolated environments for your online work, keeping your data safe and your browsing focused. Think of it as having separate, locked-down rooms inside your regular browser â€“ perfect for sensitive tasks or collaborative projects.

Key Features

Fortal Browser offers several key benefits:

  • Isolated Workspaces: Each workspace runs in its own secure container, meaning activities in one don't affect others.
  • Controlled Access: Easily restrict the websites that can be visited within a workspace (either a whitelist or a blacklist).
  • Enhanced Security: Built-in protection against unwanted pop-ups and restricted keyboard shortcuts further lock down the environment.
  • Easy File Sharing: A straightforward interface for uploading and downloading files within the workspace.
  • Real-Time Collaboration: Enables secure sharing and real-time interaction for teams.
  • Flexible Customization: Easily tailor browsing restrictions, use extensions, and modify appearance to align perfectly with your use case.
  • Admin-Friendly Interface: Easy access and management for admin users through a separate client.

Usage Guide

Once set up, using Fortal Browser is easy:

Accessing Workspaces

An administrator will provide you a link to the workspace portal. To access the portal, open the URL in any standard web browser.

Login

The administrator will provide login credentials. Enter your Display Name, Password, and Email Address to get into the workspace.

Navigating the Dashboard

  • Overview: Provides a high-level summary of the workspace.
  • Files: Enables uploading/downloading files securely within the workspace.
  • Chats: A communications hub to interact and collaborate.
  • Customization: Within the workspace dashboard, you can customize the interface to your liking. The Fortal Browser lets you adjust settings like browser profiles, the theme of the new tab, and file management.

Overview

The landing page displays a brief description of the project, its creation and end date, and the number of freelancers working on the current project.

File Section

Permits users to upload their files to the current project and download files uploaded by others.

Chats Section

Users can send a quick chat to the administrator.

Secure Browsing

Once inside a workspace, you can be sure that it is secured, and Fortal is protecting your activities!

Troubleshooting & FAQs

Problem: I can't access a particular website.

Solution: The website might be blocked by the workspace's restrictions. Contact your administrator if you need access to it.

Question: Can I use my normal browser extensions?

Answer: Fortal Browser has a customized Firefox profile, which makes the browser more secure. If the "Use Extensions" option is selected, then the Fortal add-ons will be used.

Contact a Fortal.io Representative: For feature requests, please contact your administrator to relay it to a Fortal.io expert.
